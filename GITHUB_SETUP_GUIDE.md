```
┌─────────────────────────────────────────────────────────────┐
│  Create a new repository                                     │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Repository name *                                          │
│  ┌──────────────────────────────────┐                      │

│                                                             │
│  Public  ○ Private                                          │
│    ☑️ ← اختر Public                                         │
│                                                             │
│  ☐ Add a README file (اتركه فارغاً)                         │
│  ☐ Add .gitignore (اتركه فارغاً)                           │
│  ☐ Choose a license (اتركه فارغاً)                          │
│                                                             │
│         [Cancel]           [Create repository]              │
│                           ← اضغط هنا                        │

```
