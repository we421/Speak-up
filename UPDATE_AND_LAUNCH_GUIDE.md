## 🎯 كيف تبدأ الآن

### الخطوة 1: نشر التطبيق على Vercel

```bash
# 1. Commit التغييرات
git add .
git commit -m "Add gamification, curriculum, and marketing content"

# 2. Push إلى GitHub
git push origin main

# 3. افتح Vercel في المتصفح
# https://vercel.com

# 4. اتبع الخطوات:
# - Login بـ GitHub
# - Add New → Project
# - اختر "Speak-up"
# - اضغط Deploy
# - انتظر 2-5 دقائق
# - احصل على رابط مثل: https://speakup.vercel.app
```

### الخطوة 2: تهيئة المنهج التعليمي

بعد الحصول على رابط Vercel، افتحه في المتصفح:

```bash
# استبدل YOUR_VERCEL_URL برابطك الفعلي
curl "https://YOUR_VERCEL_URL.vercel.app/api/curriculum?action=initialize"
```

### الخطوة 3: تحديث الروابط في ملفات التسويق
