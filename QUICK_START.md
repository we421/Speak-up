# 🚀 البدء السريع - English Master

## الخطوة 1: تشغيل التطبيق الآن على جهازك

### المتطلبات:
- Node.js (18 أو أحدث) - حمّله من: https://nodejs.org
- Bun أو npm

### التشغيل السريع:
```bash
# 1. تثبيت المكتبات
bun install
# أو: npm install

# 2. تشغيل قاعدة البيانات
bun run db:push

# 3. تشغيل التطبيق
bun run dev

# 4. افتح المتصفح على:
http://localhost:3000
```

---

## الخطوة 2: تثبيت التطبيق كـ PWA

### على الكمبيوتر (Chrome/Edge):
1. افتح http://localhost:3000
2. ابحث عن أيقونة التثبيت 📱 في شريط العنوان
3. انقر "Install" أو "تثبيت"

### على Android:
1. افتح http://localhost:3000 في Chrome
2. القائمة (ثلاث نقاط) ⟡
3. "Install App"

### على iPhone:
1. افتح http://localhost:3000 في Safari
2. زر المشاركة ↑
3. "Add to Home Screen"

---

## الخطوة 3: تخصيص التطبيق باسمك

### 1. غيّر الاسم في `public/manifest.json`:
```json
{
  "name": "اسم تطبيقك هنا",
  "short_name": "الاسم المختصر"
}
```

### 2. غيّر الاسم في `src/app/layout.tsx`:
```typescript
export const metadata: Metadata = {
  title: "اسم تطبيقك - الوصف",
  // ...
};
```

### 3. أنشئ أيقونة مخصصة:
- اذهب إلى: https://realfavicongenerator.net
- ارفع صورتك (512x512 بكسل)
- حمل الملفات وضعها في `public/icons/`

### 4. غيّر الألوان:
```json
// في manifest.json
{
  "theme_color": "#لونك_المفضل"
}
```

## الخطوة 4: نشر التطبيق مجاناً (Vercel)

### الطريقة الأسهل (موصى بها):
1. اذهب إلى: https://vercel.com
2. سجل حساب مجاني
3. Connect with GitHub
4. ارفع كودك على GitHub
5. Import Project في Vercel
6. Deploy!

### الخطة المجانية تشمل:
- ✅ نطاق مجاني: your-app.vercel.app
- ✅ نطاق مخصص مجاني (إن وجد)
- ✅ HTTPS تلقائي
- ✅ CDN عالمي
- ✅ تحديثات تلقائية من GitHub

## الخطوة 5: البدء في الربح

### 1. Google AdSense (الأسرع):
```
1. سجل: https://adsense.google.com
2. أضف موقعك
3. احصل على كود الإعلان
4. أضفه في التطبيق
```

### 2. اشتراكات Premium:
```
1. أضف Stripe للدفع
2. أنشئ خطط: مجاني / برو
3. الميزات المدفوعة:
   - محادثة غير محدودة
   - جميع الدروس
   - تتبع متقدم
```

## الخطوة 6: النشر على متاجر التطبيقات (اختياري)

### Google Play:
1. حساب مطور ($25)
2. استخدم PWABuilder.com للتحويل
3. ارفع APK إلى Play Console

### App Store:
1. حساب مطور ($99/سنة)
2. Mac + Xcode مطلوب
3. مراجعة أطول وأكثر صرامة
